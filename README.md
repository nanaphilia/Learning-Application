# Learning Application

A Flutter-based educational application developed as a learning project. Students can review learning modules and complete quizzes to evaluate their understanding.

## Features
- User authentication
- Module-based lessons
- Interactive quizzes
- Progress tracking

## Technologies Used
- Flutter
- Dart
